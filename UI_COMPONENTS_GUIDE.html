6. Transition & Timing

--transition-fast: 150ms cubic-bezier(0.4, 0, 0.2, 1)

--transition-smooth: 300ms cubic-bezier(0.4, 0, 0.2, 1)

--transition-bounce: 400ms cubic-bezier(0.34, 1.56, 0.64, 1)

Pro Tip: Use fast for micro-interactions (hover, focus), smooth for page transitions, bounce for celebratory effects

7. Shadow System

Elevation levels from subtle to prominent:

  • --shadow-xs: Minimal (1px blur)
  • --shadow-sm: Subtle (2px blur)
  • --shadow-md: Medium (6px blur, default cards)
  • --shadow-lg: Large (15px blur, buttons on hover)
  • --shadow-xl: Extra Large (25px blur, elevated cards)
  • --shadow-2xl: Maximum (50px blur, modals, hero sections)
